** アイテム一括検索アプリ 仕様書 **

* 名前からIDを一括で探したいので作りました
* AIで作成しました
* pythonは3.13を使っています
* 開発者がいなくなっても利用者が簡易な手順でアップデートしながら使い続ける事を目標にしてます

* 日本語名で複数のアイテムを一括で検索し、IDをコピーする機能を持つアプリです
* コピー リセット機能 部分一致 完全一致 機能があります
* item_list.json が必要です(NLP\item_list.json)

* item_list.json の編集で追加削除も容易です

* Cドライブ直下に c:\Python313\build\itemlist というフォルダを作ってください

* cd /d C:\Python313\build\itemlist をコマンドプロンプトで実行し、フォルダ指定して下さい

* GUIを使うために "customtinker"というpythonでGUIアプリを作れるようになる拡張機能を導入します

** コンパイル **
0. アプリに組み込むので item_list.json を同じフォルダに置いて下さい

1. customtinker インストール
コマンドプロンプトを立ち上げる

<<この次の行をコピーする>>
pip3 install customtkinter
<<ここまで>>

上記文字列をコピーしてコマンドプロンプトで実行する

2. customtinker アップグレード
頻繁な更新があるためアップグレードが推薦されています

<<この次の行をコピーする>>
python.exe -m pip install --upgrade pip
<<ここまで>>

上記文字列をコピーしてコマンドプロンプトで実行する

3. PyInstaller のインストール
実行ファイル(exe)化するためのmodです
まず、EXE化するためのツールをインストールします。ターミナル（コマンドプロンプト）で以下のコマンドを実行してください。

<<この次の行をコピーする>>
pip install pyinstaller
<<ここまで>>

4. search_item_list.py を Python313\build\itemlistフォルダに配置する(他のフォルダでも可能)
<<コマンドプロンプト>>
cd c:\Python313\build\itemlist
<<コマンドプロンプトここまで>>

5. コンパイルして実行ファイルを作る

<<この次の行をコピーする>>
pyinstaller --noconsole --onefile --add-data "item_list.json;." search_item_list.py
<<ここまで>>

distというフォルダにsearch_item_list.exeが作られます

6. 起動します

** 使い方 **

* 一番上の検索窓にアイテム名を入力して下さい
* 複数検索に対応しているので改行して入力して下さい
* 検索結果が下に表示されます
* IDコピーは真ん中のボタンです
* リセットで初期化します
* スロット数の [0] [1] [2] [3] [4] や (上段) [Str + 1] と [Str +10] などの検索は部分一致で複数出力されます
* 武器防具を完全一致で検索する場合は ナイフ[0] コットンシャツ[1] のようにスロット数の接尾語を追加して下さい
* 半角スペース・アルファベットに注意して下さい(Str + 1 は STR + 1 Str+1 Str +1 str + 1 どれも不可)
* ブラウザからコピーすると html の改行コードが見えない状態で残るので、その辺のテキストエディタで消して下さい
